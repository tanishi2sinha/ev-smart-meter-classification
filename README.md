# EV Smart Meter Classification
Machine learning models for identifying electric vehicle charging behaviour from smart meter data using feature engineering and classification algorithms.

## Overview
This project uses machine learning techniques to identify electric vehicle (EV) charging behaviour from household smart meter data. Multiple classification models were developed and compared to determine the most effective approach.

## Features
- Data preprocessing
- Feature engineering
- Machine learning model training
- Model performance evaluation

## Machine Learning Models
- Logistic Regression
- K-Nearest Neighbours (KNN)
- Support Vector Machine (SVM)

## Dimensionality Reduction
- Principle Component Analysis

## Results
- KNN achieved the best overall classification performance.
- Models were evaluated using accuracy, precision, recall, F1-score, and confusion matrices.

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
